Overview

SmartMirror is a modern, interactive display system designed specifically for barbershops. It combines appointment management, customer recognition, and real-time information to create a premium experience for both barbers and customers.

Try It Yourself

1 View the Mirror Display

Visit barbermirror.replit.app to see what your customers would see:

  • Clock & Weather - Real-time display in top corners
  • Messages Panel - Shows Telegram messages and announcements
  • Appointments - Today's scheduled appointments
  • News Feed - Scrolling headlines
Try it: Click "Detect Face" or "Voice Command" and say "Mirror mirror, what time is it?"

2 Explore the Admin Panel

Visit barbermirror.replit.app/admin

Login: Username: admin Password: admin123
  • Budget - Track daily earnings, set revenue goals
  • Services - Add/edit haircut services and prices
  • Telegram Bot - See bot status and recent messages
  • Mirror Controls - Send messages to the mirror display
  • Modules - Enable/disable mirror features
  • Team - Manage barbers (Admin only)
  • Mirrors - Configure multiple mirror devices

3 Test the Telegram Bot

Open Telegram and search for @BarberMirrorBot

Booking flow: Book Appointment → Select service → Choose barber → Pick time → Confirm. Watch it appear on the mirror!
Send a message: Tap "Send Message to Mirror" → Type your message → See it display instantly

Key Features

Interactive Mirror Display

Real-time clock, weather, appointments, news, and custom messages from management.

Face Recognition

Personalized greetings when customers approach. Privacy-focused with self-registration via QR code.

Voice Commands

Hands-free interaction: "Mirror mirror, show appointments" or "what's the weather?"

Telegram Bot

Manage your shop from your phone - book appointments, track earnings, send messages.

Multi-Location Support

Separate data for each barbershop with multiple mirrors and individual barber accounts.

Budget Tracking

Set daily/weekly goals, track earnings per barber, monitor shop performance.

Deployment Options

SmartMirror is cloud-hosted, so your display hardware just needs a web browser.

Hardware Cost Best For
Android TV Box $40-60 Most shops - affordable, reliable
Amazon Fire TV Stick $30-50 Budget option - plug and play
Orange Pi 5 $60-90 Tech-savvy owners
Mini PC / Intel NUC $150-300 Premium installs
Repurposed Laptop Free Testing or budget shops

What You Need

  • Display - Any TV or monitor (32" or larger recommended)
  • Display device - Android box, Fire Stick, or mini PC
  • Internet - WiFi or ethernet connection
  • Camera (optional) - USB webcam for face recognition
  • Speaker (optional) - For voice greetings
